## About

An LLM-based translation and parsing framework designed to bridge the gap between Natural Language instructions and the CoBlock formal language rules.

## Usage

Prerequisites
```sh
git --version
python --version
pip --version
```

Installation & Usage
```sh
git clone https://github.com/lilliban/NL-to-CoBlock
cd NL-to-CoBlock
python -m venv .venv # creates the virtual environment
source .venv/bin/activate # start session
pip install -r requirements.txt # add all the required libraries
python3 -m src.main
```

> [!IMPORTANT]
> Remember to end the session once you are done working with `deactivate`
